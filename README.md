# Gankoo
==========

API from [gank.io: https://gank.io/api.](https://gank.io/api)

业余的iOS项目，使用Swift3编写
 

## Feature

- 纯 Swift 项目  

- 数据存储使用的是Realm

- 支持CoreSpotlight
